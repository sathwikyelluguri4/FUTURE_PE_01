# Prompt Documentation

## Task

AI Website Copy Generator for Local Businesses

## Business

Urban Brew Café

## Prompt 1

Act as an expert website copywriter. Create a modern, engaging homepage copy for a local café called Urban Brew Café. Include a hero section, value proposition, about section, and welcoming tone.

## Prompt 2

Generate a professional services page for Urban Brew Café. Include coffee varieties, breakfast menu, snacks, desserts, takeaway, and free Wi-Fi.

## Prompt 3

Write three persuasive Call-to-Action sections encouraging visitors to visit the café, book a table, or contact the business.

## AI Tool Used

ChatGPT
